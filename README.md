# 📊 End-to-End E-Commerce Sales Analytics

An end-to-end Data Analytics project built using the **Brazilian Olist E-Commerce Dataset**. This project demonstrates the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, SQL database design, business analysis, and interactive Power BI dashboard development.

---

## 🚀 Project Overview

The objective of this project is to analyze an e-commerce business and uncover meaningful business insights related to:

- Revenue Analysis
- Customer Behavior
- Product Category Performance
- Payment Analysis
- Delivery Performance
- Order Trends

The project follows a complete analytics pipeline from raw data to business dashboard.

## 📂 Dataset

**Dataset:** Olist Brazilian E-Commerce Public Dataset

The dataset contains approximately:

- 100K+ Orders
- 99K Customers
- 32K Products
- Multiple relational tables including:

- Customers
- Orders
- Order Items
- Payments
- Products
- Reviews

## 💡 Key Business Insights

- Beauty and Health products generated the highest revenue.
- Weekday purchases contributed significantly more revenue than weekends.
- Credit Card was the most frequently used payment method.
- Average delivery time was approximately 12 days.
- Customer purchasing behavior varied across different product categories.
